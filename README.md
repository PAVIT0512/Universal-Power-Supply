## Components

### Transformer
The transformer steps down the high-voltage AC from the mains to a lower, more manageable AC voltage. It also provides electrical isolation between the high-voltage input and the low-voltage output, enhancing safety. ğŸ 

### Rectifier
The rectifier converts the AC voltage from the transformer into pulsating DC. This process changes the alternating nature of AC to a unidirectional flow of current. The most common configuration is the bridge rectifier, which uses four diodes for full-wave rectification, efficiently utilizing the entire AC cycle. ğŸ”„
